# Liberty Desktop Pet for Windows

Windowsの画面上で動く、リバティアイランド風のピクセル馬ペットです。
枠なし・透明背景・常に手前表示で動きます。

## 起動

`start.bat` をダブルクリックします。

Pythonなどの追加インストールは不要です。Windows標準のPowerShellで動きます。

## 操作

- ドラッグ: 場所を移動
- クリック: 頭を撫でる
- ダブルクリック: すぐ肉を食べる
- 右クリック: 終了
- `q` または `Esc`: 終了
